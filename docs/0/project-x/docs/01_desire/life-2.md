# Life: 生存の論理と実相

本プロジェクトの究極目的である「人生の価値の最大化」を定義するために、その対象である「人生（人が生きること）」の本質について定義する。

## 1. 世界と存在 (World and Existence)

この世界は物理法則によってのみ構成されている。そこに善悪や意味といった概念は存在しない。ただ物質があり、法則に従って運動しているだけである。

石や水といった非生物は、ただそこに「ある（存在）」。彼らは自己を維持しようとする機能を持たず、外部環境の変化に従って崩壊・変質していく。彼らにとって、存在し続けることも消滅することも、等しく物理現象の結果に過ぎない。

この物理世界において、自らの形状と機能を維持しようと外部環境に働きかける特異な物理現象が発生した。これを「生存」と呼び、その主体を「生物」と呼ぶ。生存とは、エントロピー増大の法則に抗い、自己という秩序を保とうとする物理的な抵抗運動である。

## 2. 自己認識と痛みの非対称性 (Self-Recognition and Asymmetry of Pain)

生物が生存するためには、自己を維持するためのリソースを外部から取り込まなければならない。そのためには、まず「自己」と「他者（外界）」を区別する必要がある。

この区別の根源にあるのは「痛み」である。
自分の体が傷つけば痛みを感じるが、他者の体が傷ついても痛みは感じない。この感覚の非対称性が、物理的には連続した物質の世界において「ここから内側が自分である」という境界線を引く。

この認識により、生物は「自分だけは特別である」という錯覚（主観）を獲得する。自己の保存こそが世界の中心であり、他者はそのための環境や道具に過ぎないという認識は、生存本能が正常に機能している証である。

## 3. 生存と依存 (Survival and Dependency)

### 生きるとは「依存」である
生物が自己を保存するためには、エネルギーと物質が必要である。しかし、自己の内側だけでそれらを生成し続ける永久機関は物理的に不可能である。故に、生物は外部にある物質を内部に取り込み、自己の構成要素へと変換し続けなければならない。

呼吸により酸素を、飲水により水を、捕食により他者の血肉を取り込む。自己とは、取り込まれた他者の集合体であり、物理的な境界線は極めて曖昧である。生きるとは、恒久的な他者依存プロセスである。

### 「自立」という欺瞞
物理法則上、他者に依存せず生きることは不可能である。しかし、依存とは「依存先が消えれば自分も死ぬ」という恐怖と隣り合わせの状態である。この恐怖に常に晒されれば、精神はパニックに陥り、合理的な生存判断ができなくなる。

故に人間は、生存率を高めるために事実を認知から排除し、「自分は依存していない（自立している）」という嘘の認識を持つことで精神を安定させる。「自立」とは実在しない概念であり、生存本能が作り出した自己防衛のための欺瞞である。

## 4. 善悪と自己矛盾 (Good, Evil, and Paradox)

### 善悪の起源
生物は生存ルートを迅速に選択するために、自己の生存に有利なものを「善」、不利なものを「悪」と分類する主観的な判断機能（利己）を持つ。生物にとっての至上命題は「生存」であり、それこそが根源的な「善」である。

### 生きるとは「悪」である
しかし、生存（依存）の実態は、他者の存在や構造を破壊し、奪うことである。特に他生物への依存は、他者にとっての至上命題である「生」を否定し、殺害することを意味する。

ここに論理的な矛盾が生じる。
自らの「生（善）」を成立させるためには、他者の「生（善）」を否定（悪）しなければならない。
至上命題であるはずの「生」を、生きるために自ら否定せねばならない。

理性を持つ人間は、この因果関係を理解できてしまうが故に、生きることは悪であるという結論に到達せざるを得ない。本能は「生きろ」と命じ、理性は「それは悪だ」と告げる。この逃れられない葛藤こそが、人生の苦痛の根源である。

## 5. 生存戦略としての偽り (Strategy of Deception)

正面からの奪い合いは、自己が殺害されるリスクを高める非合理な行為である。故に生物は、生存率を最大化するために「偽り、騙し、奪う」という戦略を採用する。

1.  **偽る:** 自分を大きく見せ、あるいは無害に見せかけ、前提を歪める。
2.  **騙す:** 他者の警戒心を解き、抵抗の機会を奪う。
3.  **奪う:** 他者を弱体化させ、支配下に置く。
4.  **殺す:** リスクを負わずに一方的に命を絶つ。
5.  **食う:** 他者を自己の一部（エネルギー）に変える。
6.  **生きる:** 1〜5を繰り返す。

### 悪の証明としての合理性
偽り、騙し、奪うことは、殺害（生の否定）を確実にするために実行される計画的なプロセスである。
これらの行為が生存にとって「合理的」であればあるほど、生きるという行為が「生を否定する（悪である）」という事実は、より強固に裏付けられる。

生存本能に従い、より良く生きようとすればするほど、他者を陥れる手口は洗練され、悪の度合いは深まっていく。
生きるとは、この悪の論理を自らの行動によって証明し続けることに他ならない。

## 6. 結論 (Conclusion)

生きるとは依存であり、殺害であり、悪である。
人間は、誕生という受動的な運命によってこの世界に投げ出され、生存本能というプログラムによって、罪悪と苦痛に満ちた生を継続することを強制されている。

この救いようのない現実（人生）の中で、それでもなお生存を続けるためには、生きるという罪悪と苦痛を正当化するための「根拠」が必要となる。

## 6. 結論 (Conclusion)

生きるとは依存であり、殺害であり、悪である。
人間は、誕生という受動的な運命によってこの世界に投げ出され、生存本能というプログラムによって、罪悪と苦痛に満ちた生を継続することを強制されている。

理性を持つ個体にとって、この「生きることは悪である」という自己矛盾を、たった一人で抱え続けることは精神的な限界を超える。
故に人間は、この罪悪感と責任を分散させ、正当化するための仕組みを求めた。自分以外の誰かに責任を転嫁し、集団の中に埋没することで、自己否定の苦痛から逃れようとしたのである。

その逃避と依存の果てに構築されたのが、「社会」という巨大な欺瞞システムである。
