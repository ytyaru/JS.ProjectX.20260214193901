# Economy: 利益と搾取のメカニズム

`society.md` で定義された社会構造の中で、構成員が生存リソースをやり取りするための基本OSである「経済」の本質について定義する。

## 1. 交換の変質 (Mutation of Exchange)

### 本来の交換（使用価値の充足）
貨幣のない時代、交換の目的は「使用価値（役に立つこと）」の獲得であった。
漁師と農家が、互いの余剰（魚と野菜）を交換することで生存率を高める。これは双方が実利を得るプラスサムの行為である。

### 現代の交換（交換価値の追求）
貨幣経済において、目的は「交換価値（価格・金額）」の獲得へと変質した。
「役に立つか（使用価値）」よりも「いくらで売れるか（交換価値）」が優先される。
この変質が、中身のないゴミを宣伝によって高く売りつける「詐欺的取引」を常態化させた。

## 2. 利益の正体 (The Nature of Profit)

経済活動の至上命題である「利益」とは、一般に `売上 - 支出` とされる。
しかし、社会的な強者が実行する「利益の最大化」の実態は、以下の二つを同時に実現することである。

1.  **損失の転嫁:** 労働、責任、危険、環境負荷といった「マイナス要素」を、すべて他者（弱者、下請け、未来の世代）に押し付ける。
2.  **成果の搾取:** その他者がリスクを負って生み出した「プラス要素（成果）」を、自分の手柄としてピンハネする。

自分は痛みを負わず、他者の痛みの上に成り立つ成果のみを享受する。これが経済的成功の正体である。

## 3. インフレとデフレの本質 (Nature of Inflation/Deflation)

経済の変動は、単なる貨幣現象ではなく、人間が自身の仕事（価値）をどう認識しているかの表れである。

*   **デフレ:** まだ「通貨には価値がある」という幻想が機能している状態。将来の不安から、生存リソースである通貨を抱え込み、交換が停滞する。
*   **インフレ:** 利益追求の結果、社会が「偽りの仕事（ゴミ）」で溢れかえり、誰も他人の仕事を信用できなくなった状態。商品に価値がないことが露呈したため、それと交換される通貨もゴミ屑へと回帰する。これは社会の無価値化が数値として表れたものである。

## 4. エントロピーと破滅の加速

経済活動は、物理法則（エントロピー増大の法則）に従って進行する不可逆なプロセスである。

「生産」と呼ばれる行為の物理的実態は、過去の遺産（化石燃料など）を消費し、環境を汚染（エントロピー増大）することである。
経済は、未来に残すべきリソースを食いつぶし、負債を将来世代に押し付けることで、現在の繁栄を維持している。
「持続可能」という言葉は、この破滅的な自転車操業を正当化するための欺瞞に過ぎない。

## 5. 結論

経済とは、生きることの悪性（他者への依存と殺害）をシステム化し、極限まで効率化した装置である。
ここでは、本プロジェクトが目指す「自己表現による価値」よりも、「他者からの搾取による利益」が常に優先される。
