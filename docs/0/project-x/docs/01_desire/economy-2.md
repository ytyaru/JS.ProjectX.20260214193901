# Economy: 利益と搾取のメカニズム

`society.md` で定義された社会構造の中で、構成員が生存リソースをやり取りするための基本OSである「経済」の本質と、それがもたらす必然的な破綻（メタクソ化）について定義する。

## 1. 交換の変質と支配の構造 (Mutation and Domination)

本来、交換とは「生存の実利」を得るための行為であった。マルクス経済学で言う **「使用価値 (Use Value)」** の獲得である。
漁師と農家のように、自らの生存のために生産した余剰を交換し合うことは、相互に生存率を高める行為である。

しかし、現代社会において、この目的は **「交換価値 (Exchange Value)」**、すなわち「利益」の獲得へと変質した。
中身のないゴミを宣伝によって高く売りつける詐欺的取引が常態化している。なぜ、このような非生産的な活動で生計を立てることが可能なのか？

それは、この経済圏の外側に、圧倒的な暴力によって支配され、生存リソースを安価に奪われている「奴隷（途上国や社会的弱者）」が存在するからである。
戦争や植民地支配によって他者の命と時間を極限まで買い叩き、そこで生じた莫大な余剰リソースの上で、初めて「ゴミを売り合う」という虚構のゲームが成立する。
使用価値から交換価値への移行は、群れが社会へと巨大化し、他者を組織的に犠牲にする生存戦略を確立したことの証明である。

## 2. 利益の正体 (The Nature of Profit)

経済活動の至上命題である「利益」とは、一般に `売上 - 支出` とされる。しかし、この式は欺瞞である。
社会的な強者が実行する「利益の最大化」の実態は、以下の計算式で表される。

> **利益 ＝ 損失の転嫁 × 成果の搾取**

1.  **損失の転嫁:** 労働、責任、危険、環境負荷といった「マイナス要素」を、すべて他者（弱者、下請け、未来の世代）に押し付ける。
2.  **成果の搾取:** その他者がリスクを負って生み出した「プラス要素（成果）」を、自分の手柄としてピンハネする。

これらは相互に作用し、搾取の効率を高める。自分は痛みを負わず、他者の痛みの上に成り立つ成果のみを享受する。これが経済的成功の正体である。

## 3. インフレとデフレの本質 (Nature of Inflation/Deflation)

経済の変動は、貨幣現象ではなく、生物としての人間が「社会」という環境をどう認識したかの結果である。

*   **デフレ:**
    「貨幣を持っていれば生存できる」という幻想がまだ機能している状態。将来への不安から、生存リソースである貨幣を抱え込み、交換（他者との関わり）を拒絶する。
*   **インフレ:**
    「この社会の人間は、自分を生かすどころか奪う敵である」という事実を、本能・理性・感性のすべてが理解してしまった状態。
    「経済的合理性」の名の下に行われる搾取を肌身で感じ、他者の提供する労働や商品に価値がないことが露呈する。結果、それと交換される通貨もゴミ屑へと回帰する。これは「信用」という欺瞞が剥がれ落ち、社会の無価値化が数値として表れたものである。

## 4. エントロピーと破滅の加速 (Entropy and Acceleration)

経済活動は、物理法則（エントロピー増大の法則）に従って進行する不可逆なプロセスである。

「生産」と呼ばれる行為の物理的実態は、過去の遺産（化石燃料など）を消費し、環境を汚染（エントロピー増大）することである。
経済は、有限なリソースを食いつぶし、負債を将来世代に押し付けることで、現在の繁栄を維持している。

「未来のために」と嘯きながら、実際には「今、自分が生き残るために」未来を食いつぶす。
他者に依存せねば生きられぬほど弱体化した現代人は、この破滅的な自転車操業を止めることができない。事実を偽ることでしか自己を正当化できず、その欺瞞が理性を麻痺させ、自滅を確定させる。

## 5. 結論：悪の増幅装置 (Amplifier of Evil)

生きることは依存であり、殺害であり、悪である。
その生存本能を持つ人間が群れ、社会を形成し、経済というツールを用いた結果、必然的に悪（他者への寄生と搾取）はシステム化され、増幅された。

誰かが悪意を持って設計したわけではない。誰もが「生きたい」と願い、最適化を目指した結果、社会は「悪の増幅装置」となった。
ここでは、本プロジェクトが目指す「価値」よりも、「他者からの搾取による利益」が常に優先される。

このシステムの上で、真の「価値」を語ることは不可能である。
我々は、この沈みゆく船の中で、互いに奪い合いながら溺れるのか。それとも、この欺瞞のゲームから精神的に離脱し、自らの手で「生きる根拠（価値）」を創出する道を選ぶのか。
