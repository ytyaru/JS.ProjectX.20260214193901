# Life: 生存の論理と実相

本プロジェクトにおける「価値」を定義するための前提として、生物としての「生きること」の本質、そこから派生する認識の矛盾、そして社会構造の必然的な帰結について定義する。

## 1. 世界と存在 (World and Existence)

この世界は物理法則によってのみ構成されている。そこに善悪や意味といった概念は存在しない。ただ物質があり、法則に従って運動しているだけである。

石や水といった非生物は、ただそこに「ある（存在）」。彼らは自己を維持しようとする機能を持たず、外部環境の変化に従って崩壊・変質していく。

この物理世界において、自らの形状と機能を維持しようと外部環境に働きかける特異な物理現象が発生した。これを「生存」と呼び、その主体を「生物」と呼ぶ。生存とは、エントロピー増大の法則に抗い、自己という秩序を保とうとする物理的な抵抗運動である。

## 2. 生存と依存 (Survival and Dependency)

生物が自己を保存するためには、エネルギーと物質が必要である。しかし、自己の内側だけでそれらを生成し続ける永久機関は物理的に不可能である。故に、生物は外部にある物質を内部に取り込み、自己の構成要素へと変換し続けなければならない。

呼吸により酸素を、飲水により水を、捕食により他者の血肉を取り込む。自己とは、取り込まれた他者の集合体であり、物理的な境界線は極めて曖昧である。「自立」という概念は、この依存関係が見えにくくなった状態を指す言葉に過ぎず、物理的実態としては、生きることは恒久的な他者依存プロセスである。

生物は、依存する主体である。誕生という現象自体、親という他者の交尾の結果であり、そこに本人の意志（主体性）は介在しない。生物は、自らの意志で生きたいと願ったわけではなく、誕生した瞬間に備わっていた「生存本能」というプログラムによって、生きざるを得ない状態に置かれているに過ぎない。

## 3. 善悪と自己矛盾 (Good, Evil, and Paradox)

生物は生存するために、自己と他者を区別する認識機能を持つ。そして、生存ルートを迅速に選択するために、自己の生存に有利なものを「善」、不利なものを「悪」と分類する主観的な判断機能（利己）を持つ。生物にとっての至上命題は「生存」であり、それこそが根源的な「善」である。

しかし、生存（依存）の実態は、他者の存在や構造を破壊し、奪うことである。特に他生物への依存は、他者にとっての至上命題である「生」を否定し、殺害することを意味する。

ここに論理的な矛盾が生じる。
自らの「生（善）」を成立させるためには、他者の「生（善）」を否定（悪）しなければならない。
至上命題であるはずの「生」を、生きるために自ら否定せねばならない。

理性を持つ人間は、この因果関係を理解できてしまうが故に、生きることは悪であるという結論に到達せざるを得ない。本能は「生きろ」と命じ、理性は「それは悪だ」と告げる。人間は「生きたい」わけではない。本能により生きざるを得ず、理性によりそれを否定せずにはいられない。この逃れられない葛藤こそが、人生の苦痛の根源である。

## 4. 生存戦略としての偽り (Strategy of Deception)

正面からの奪い合いは、自己が殺害されるリスクを高める非合理な行為である。故に生物は、生存率を最大化するために「偽り、騙し、奪う」という戦略を採用する。自分を強く見せる、あるいは無害に見せることで相手の認識を歪め、抵抗の機会を奪い、一方的に搾取する。これは悪意ではなく、最適化された生存機能である。

人間においては、この「偽り」は自己の内面にも向けられる。
「生きることは悪である」という事実を直視し続ければ、自己否定により精神が崩壊し、生存が脅かされる。故に脳は事実を歪め、「自分は正しい」「これは正義である」「仕方がないことだ」と認識を改竄する。

自己を正当化し、他者に責任を転嫁し、悪を隠蔽する。この欺瞞こそが、理性を持ちながら生存本能を全うするための、人間の主要な生存戦略である。

## 5. 社会の構造 (The Structure of Society)

人間社会は、個人の生存戦略（依存と偽り）が集積し、システム化されたものである。

経済活動における「利益」とは、売上から支出（損失）を引いた残りである。利益を最大化し、確実にするための最善手は、労働や責任という「損失（リスク）」を他者に押し付け、その成果のみを奪う（ピンハネする）ことである。

社会的な強者とは、この搾取の構造を構築し、他者を効率的に利用できる者のことである。彼らは「自立」や「貢献」という美辞麗句（偽り）を用いて他者を扇動し、自らの手足として利用する。

営利目的で運営されるシステムは、必然的に「メタクソ化 (Enshittification)」する。利益（搾取）が目的化すれば、サービスや商品の品質（コスト）は削ぎ落とされ、無価値なものを価値あるものに見せかける偽装工作（広告）にリソースが割かれるからだ。これは誰かの悪意による事故ではなく、生存本能に従って利益を追求した結果の、必然的な帰結である。

## 6. 結論：価値の必要性 (Conclusion)

生きるとは依存であり、殺害であり、悪である。
社会は欺瞞に満ちており、他者に依存すれば搾取され、加担することになる。

この救いようのない現実の中で、それでもなお生存を続けるためには、生きるという罪悪と苦痛を正当化するための「根拠」が必要となる。
それが「価値」である。

価値とは、物理世界には存在しない幻想である。
しかし、その幻想を自らの手で創出し、「自分には生きる価値がある（生きていてもよい）」と自己自身に証明することでのみ、人は精神的な死（自己否定）を回避し、生存を全うすることができる。

価値の創出は、他者の評価（外的動機）に依存してはならない。他者の価値観はその他者の生存戦略であり、自分にとっての害悪になり得るからだ。
自分の価値は、自分で創出しなければならない。
