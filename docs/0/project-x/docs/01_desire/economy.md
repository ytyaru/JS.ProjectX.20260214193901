## 1. 交換

経済とは交換である。交換する物品には価値がある。マルクス経済学ではこれを使用価値・交換価値と呼ぶ。それぞれの価値から経済について説明する。

### 使用価値

経済のはじまりは物々交換だ。農家と漁師が野菜と魚を交換する。これにて栄養補完され互いの生存率が向上し両者両得の状態が成立する。これが経済の起源であり価値だった。

使用価値とは人が物品を使用することで発生する価値のことである。交換する物品は使用価値があるものだ。使用価値なきものを交換する動機はないため交換は成立しない。価値とは、人間が物品に見出す主観的な認識である。人間にとって価値なき物品は交換されず、そこに使用価値はない。

人間は使用価値あるものしか作らない。なぜなら生活の中で価値なきものを生産する余裕も動機もないからだ。こうした生活の中で、使用価値はあるが自分一人や周囲の人々では使い切れないものに関しては、市場に出して他の物品と交換することで、より豊かな生活ができる。この時、はじめて交換する動機が生まれる。交換するほどの余剰がある時、はじめて交換可能なのである。

経済とは、他者に渡す余剰分がある時のみ可能な取引である。もし余剰がなければ、そもそも交換は発生しない。他者に渡し交換せず全て自分で消費するからだ。これが地産地消であり自給自足だ。

### 交換価値

他者に渡す余剰分が増えると、交換が頻繁に行われることが当たり前になった。このとき、交換する行為にコストやリスクがあることに気づく。物品を移動させるコスト、物品が腐敗し低品質化するリスクなどである。なるだけ交換するコストやリスクを最小化するために、通貨という交換媒体を開発した。

通貨の実体は硬貨であり金属としての価値をもって交換する物品に値するものであるものとした。通貨は金属だから物理的に腐敗することもない。移動するにも小さいため、重い物品よりも低コストで運べる。これらの合理性・効率性から、人々は物々交換から通貨交換へ遷移した。

通貨を使用し交換するようになると、次第に自分で物品を生産せずとも、誰かが生産した物品を別の誰かに売ることで余剰を得ることができると気づく者が現れた。

この時、この物品はいくらで売れるか？　という指標が交換価値である。

交換価値とは、自分で物品を生産しない者が、他者の物品を転売することで発生する余剰差分のことである。転売者は交換価値によって生計を立てる者である。

## 2. 余剰生産の根源

かつて人々は全員生産者だった。しかし経済により交換を頻発化できるほど余剰が生まれると、やがて生産者を辞めて転売者(交換者)として生計を立てることが可能となった。

では、なぜ交換できるほどの余剰を生み出せるようになったのか？　なぜ生産者を辞めて転売者(交換者)として生計を立てられる者が現れるほど、生産せずに済むようになったのか？

本来人間が生産者として労働していた分を、他に転嫁して生産できるようになったからである。

では、生産活動は誰に転嫁されたのか？　それが「化石燃料で働く機械」と「途上国の人々」である。

### 化石燃料

人類は産業革命により化石燃料を用いて仕事の自動化が可能になった。従来は人間がしていた生産活動を化石燃料で動く機械に転嫁できるようになったのである。

かつて人は自らの足で移動していたが、やがて自転車、人力車、馬車、自動車と発展していった。本来負うべき自分の労働を他者へ転嫁するようになったことで、余剰を生み出せるようになったのである。

### 途上国の奴隷

こうして余剰の力が増えると、今度はその力を利用して、その力を持たぬ他者を支配することを思いつく。機械を使って余剰力により他者を攻撃する。

戦争により植民地化し、そこに生きる人々を労働力として自分の手足のように働かせる。こうして余剰から更なる余剰を生み出すことに成功した。

### 余剰＝燃料×奴隷

かくして人類は、化石燃料と途上国の奴隷達を余剰力として運用する、現代の経済社会を確立したのである。

のちにこの転嫁が、発展国における社会問題となって自らに突き付けられることとなる。

## 3. 利益

経済とは利益を追求する活動である。利益とは他者の生き血だ。

人は自己保存のため群れて社会をなし、ついに同族達すら利用した。利益追求を合理化すると偽装工作や破壊活動になり、それが生物における生存本能の実現を破壊することになる。この自己矛盾を本能で理解した結果、無価値化・有害化・メタクソする。

### 社会内生存＝同族利用搾取

経済社会において自己保存を実現するためには、同族他者を利用し搾取する必要がある。既に経済はそうして成立している。

自然界では異種族を食うことであったが、人間社会という環境では、同族他者を食うことで自己保存が成立する。

食うために殺す。殺すために奪う。奪うために騙す。騙すために偽る。これにて自己保存が実現する。では、如何にして社会にいる同族他者を偽り騙し奪うか？　経済がそれである。

### 利益＝損失転嫁×成果搾取

経済学においては一般に`利益＝売上−支出`とされている。だがこの式は、支出が何であり、誰が支払っているかを隠蔽している。また、利益を最大化する式でもなければ、現実で運用されている式とも違う。正しくは以下である。

* 利益＝損失転嫁×成果搾取

こうして事実を偽ること自体が、既に経済社会で実行されている同族他者に対する偽り騙し奪うための一手である。

#### 損失転嫁

労働、責任、危険、環境負荷といった損失を、すべて他者（弱者、下請け、未来の世代）に押し付ける。

#### 成果搾取

他者が損失を負って生み出した成果を、自分の手柄としてピンハネする。（他者たちから奪う。社会保障でありお前たちに返ると嘯き騙し取る）

#### 相互作用

損失転嫁と成果搾取は相互に作用する。自分は痛みを負わず、他者の痛みの上に成り立つ成果のみを享受する。これにて上がるのが利益だ。これが経済的成功の正体である。利益とは他者の生き血である。

## 4. 偽装

`利益＝損失転嫁×成果搾取`である。経済社会の中で自己保存を実現するためには、利益を追求する必要がある。すなわち損失を他者に転嫁し、そこから発生した成果を搾取する必要がある。

だが、他者とて同じようにせねば生きられぬ。故にそう簡単に損失転嫁されまいと抵抗するし、成果搾取されまいと防衛する。

では、どうする？　武力・権力・知力・魅力など様々な力をもって他者を支配する。しかし正面から戦えば自分も傷つく。故に偽り騙し奪うことで自分を守りながら他者から奪うことで自己保存が成立する。

この時、`利益追求＝偽装工作＝損失転嫁×成果搾取`である。

なぜ現代社会では各産業界において偽装が頻発しているのか？　それは社会の中で自己保存するためには他者から騙し取らねばならないからだ。

### 暴かれる欺瞞＝社会問題

政治家は税金を盗む罪を犯しても逮捕されず法律さえ守らない。それは法治国家さえ破綻させる。社会のルールが守られず社会を崩壊させる。立法の立場にありながら積極的に法律を破る。

逮捕されて罰を受けるべきなのに、特権者として君臨すれば無罪放免で許される。それを見た下々の者たちは、その生存率が最高の状態に憧れ、目指し、治安は悪化し、社会は崩壊する。

「ルールは破るもの」であることが生存率を最高にするのだと学習させ、社会崩壊へと導く。社会の秩序が指導者によって破壊されている。それが現代社会である。

政治、報道、芸能、競技は彼らが犯罪を犯しても許され減刑される。国民的アイドルだった芸能人は刑事罰に相当する性犯罪をしても無罪である。業界から擁護され、警察も逮捕しない。スポーツ選手はチームメイトに暴行を奮っても監督や業界から擁護され、移籍しただけで逮捕されない。

警察や検察は税金を盗む裏金を作ろうが、無罪の人間を犯罪者に仕立てようが逮捕されない。自身が裁く側である時、自分だけは裁かれぬようにする。自己保存が至上命題であり、他者搾取が目的で群れているのだから、他者に転嫁するのは当然である。故に彼らの存在意義は幻想であり害悪であると証明される。

社会の一員として守るべき規範さえ守らない。彼らを裁かない。裁くべき立場の者たちが自己保存のため不当に運用し自己保存のため他者転嫁ている。

もはや社会は機能していない。元より利己実現のために他者を搾取するだけの者たちが社会人だ。当然の帰結である。社会は滅ぶべくして滅ぶ。

社会により合理化され増幅した他者依存は、自分たち自身の害悪を証明する現象となった。これを社会問題と呼ぶ。

人々が求める価値は、他者転嫁でしか得られない。自分で使用価値を創出できなくなった現代人の末路は、他者転嫁する加害者になることだけだ。はたしてそこに人は価値を見出すだろうか？　その答えは、人が自ら生み出した経済によって数値として示されることとなる。

## 5. 経済現象

経済の変動は、通貨価値の変動すなわちデフレやインフレとして観測される。それらは一般に通貨と物品の価値が上下しバランスが取れなくなった状態として説明される。だが、それは本質を隠蔽している。

通貨価値とは、使用価値が生産されなくなった時に喪失する。それは利益追求という経済活動をした末の必然だ。

デフレとは物の価値が下がったのではなく、使用価値を自ら生み出せない者たちにできる最後の抵抗だ。既に得た通貨という、まだ価値があると信じているものを守ることで起きる事態である。

インフレとは物の価値が上がったのではなく、使用価値を自ら生み出せない自分たち自身には価値がないことを理解できた時に起きる事態である。

## 4. 利益追求による価値喪失の必然

経済とは交換により利益を追求する活動だ。残念ながら経済は価値の喪失が約束された方法論である。

交換は使用価値があるから成立する。なのに使用価値ある物品の生産を辞める者が現れたら、人類全体において使用価値をもつ物品の絶対量が減少してしまうはずだ。すると交換する余剰がなくなり、自分で生産し自分で消費する従来の形に戻らざるを得なくなる。

では、もし自分で使用価値を生産できなければ？　他者から奪うしかない。

では、もし自分で使用価値を生産できない人間が大量にいれば？　誰もが他者から奪うようになる。

では、誰もが他者から奪う。そんな場や生活に、価値はあるか？　その価値は他でもない自分たち自身が決めるものだ。なぜなら価値とは人間の頭の中にある認識だから。

人間は、今の自分たちをどう認識している？　それを数値化したのが通貨価値＝交換価値である。

### 自己価値＝通貨価値＝交換価値＝加害行為＝害悪

`利益＝損失転嫁×成果搾取`であり、利益追求が社会における生存方法である。この時、他者に与える商品はコストを極限まで落とし低品質化したものである。それは使用価値が極限まで低下したものだ。

価値とは使用価値のことである。交換とは物品に使用価値があるから行われる動機があった。では、使用価値を損ねた物品しか作らない、作れない状態の商材、仕事、それをする人間や社会に、価値があると思うか？

使用価値だけが価値である。それを損ねることで利益を得ようとする交換価値は、むしろ損害だ。ここに交換価値の欺瞞が暴かれた。

交換価値という損害を追求した結果、互いに加害し合い、無価値化・有害化した。因果応報。なるべくしてなった結果である。

## 6. 終わりなき加害

経済活動は生物が自己保存をめざす限り終わらない。経済の実態は他者への加害である。その事実が発覚すれば、もう搾取される側として社会に所属する者はいなくなり、群れに戻るか、自然界での生存競争に戻るだろう。

生産と呼ばれる行為は、過去の遺産である化石燃料を消費し、環境汚染し、生存環境を破壊する自殺行為である。有限な資源を食いつぶし、負債を次世代へ押し付けることで、使用価値を生み出せぬ無価値な自己を保存している。自分たちが支払うべき代償を他者に転嫁している。それは自滅を加速させる行為に他ならない。

だが、止まることはできない。戻ることはできない。

もし、原発を辞めたら？　もし余剰力を手放したら？　自分たちがこれまで余剰力で他者を支配し利益を得てきたように、今度はそれを手放した自分たちが、支配され隷属させられることになるだろう。それは生存率を低下させることだ。そんな選択はできない。ましてや自分で使用価値を創出できない無能ならば、尚更他者に転嫁せずには生きられない。

こうして人類は、利益追求により損失を他者へ転嫁した結果、自己の無能化・無価値化が深まり、自己否定から自己研鑽し自己改善することすら不可能になってしまった史上最悪にして究極の害悪である。

## 7. 結論

経済は自己保存すべく行った他者への加害である。もはや自己の無能を他者に転嫁せねば生きられぬほど重度に依存し弱体化した人間には、生物としての生きる価値や資格はない。皮肉にも、自身で作り出した経済により数値化されている。社会問題として日々発現している。

偽り続ける限り、価値を損ね、生存を自ら破綻させる。至上命題たる自己保存を、自ら否定して種ごと終わるだろう。

この自滅を避けるためには使用価値の自己創出が必要だ。他者に依存し奪うのではなく、自ら使用価値を創出するのである。
