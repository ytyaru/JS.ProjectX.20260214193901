# Economy: 利益と搾取のメカニズム

`society.md` で定義された社会構造の中で、構成員が生存リソースをやり取りするための基本OSである「経済」の本質と、それがもたらす必然的な破綻（メタクソ化）について定義する。

## 1. 交換の起源と変質 (Origin and Mutation)

本来、交換とは価値の創出であった。
漁師と農家のように、自らの生存のために生産した「高品質な余剰」を交換し合うことは、自分一人では得られないリソースを獲得し、相互に生存率を高めるプラスサムの行為である。ここでは、人間が人間のために作り出すからこそ、それは価値ある物となる。

しかし、社会における交換は変質した。
社会とは他者に依存し、他者を利用するシステムである。故に、そこでの経済活動は「他者から収集（搾取）することで生計を立てる」ことが目的となる。目的が「生存に必要な使用価値」から、他者を操作するための「利益（交換価値）」へとすり替わったのである。

## 2. 利益の正体 (The Nature of Profit)

経済活動の至上命題である「利益」とは、単に売上からコストを引いた残りではない。
社会的な強者が実行する「利益の最大化」とは、以下の二つを同時に実現することである。

1.  **損失の転嫁:** 労働、責任、危険、環境負荷といった「マイナス要素」を、すべて他者（弱者、下請け、未来の世代）に押し付ける。
2.  **成果の搾取:** その他者がリスクを負って生み出した「プラス要素（成果）」を、自分のものとしてピンハネする。

自分は痛みを負わず、他者の痛みの上に成り立つ成果のみを享受する。これが経済的成功の正体であり、社会において「賢い生き方」とされるものの実態である。

## 3. 労働と疎外 (Labor and Alienation)

この経済システムは、人間の精神を深く蝕む。なぜなら、社会が求める「労働」と、生物が求める「善」が乖離しているからである。

生物にとっての「善」とは、己と種の生存に寄与することである。自分が心血を注いだ仕事が、仲間の役に立ち、生存を支える。この実感こそが、生物としての根源的な喜びであり、生きる手応えである。

しかし、利益追求を目的とする社会において、労働の本質は「搾取」と「偽装」になる。
自分が心血を注いで行う作業が、無価値なものを価値あるように見せかける詐欺であり、他者から富を奪うための略奪であるならば、そこに生物としての喜びは存在しない。あるのは、生存本能（悪）を肥大化させた支配者による、他者の道具化だけである。

支配者は搾取に悦びを見出すかもしれないが、それは社会全体（種）から見れば、リソースを食い荒らす癌細胞と同じである。一方、搾取される労働者は、自らの行いが「偽り」であることを自覚せざるを得ず、生きる意味（価値）を見失う。これが、現代社会における「人生の無価値化」の正体である。

## 4. エントロピーと破滅の加速 (Entropy and Acceleration)

このシステムは、物理法則とも矛盾しているように見えるが、実際には物理法則（エントロピー増大の法則）に従って破滅へと向かっている。

物理世界において、秩序はいずれ崩壊し、資源は枯渇する。この結末は不可避である。
本来であれば、消費を抑え、次世代にリソースを残す（延命する）ことが種の生存戦略としては合理的である。しかし、個体の生存本能は「今、自分が生き残ること」を最優先する。

「自分が死んだ後のことなど知ったことではない」。この究極の利己主義は、個体の生存戦略としては正しい。故に経済は、未来からリソースを前借りし、環境を破壊し、破滅を加速させることで、現在の利益を最大化する。
経済活動とは、エントロピー増大の速度を早め、人類という種の寿命を燃料にして、個人の寿命を燃え上がらせる行為である。

## 5. 結論：悪の増幅装置 (Amplifier of Evil)

経済とは、生きることの悪性（他者への依存と殺害）をシステム化し、極限まで効率化した「悪の増幅装置」である。
ここでは、価値（質）よりも利益（数字）が常に優先され、偽り騙し奪う能力が高い者ほど生存に適しているとされる。

このシステムの上で、真の「価値」を語ることは不可能である。
我々は、この沈みゆく船の中で、互いに奪い合いながら溺れるのか。それとも、この欺瞞のゲームから精神的に離脱し、自らの手で「生きる根拠（価値）」を創出する道を選ぶのか。
